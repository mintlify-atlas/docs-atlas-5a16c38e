## Introduction

The `Router` class simplifies the process of handling incoming requests and directing them to developer-defined code flows. It provides a powerful routing system with support for multiple HTTP methods, regular expressions, and namespace-based routing.

## Public Properties

### `$Path`

```php
public static ?string $Path = null;
```

The path of the current request. Set by `init()` or the constructor.

### `$Method`

```php
public static ?string $Method = null;
```

The HTTP method of the current request as a string (e.g., "GET", "POST").

### `$MethodBit`

```php
public static int $MethodBit = 0;
```

Bitwise representation of the HTTP method of the current request.

### `$CriteriaPrefix`

```php
public static string $CriteriaPrefix = '';
```

Prefix applied to criteria before matching. Useful for mounting routes under a common path.

### `$ClassPrefix`

```php
public static string $ClassPrefix = '';
```

Prefix applied to class names when resolving string callbacks.
