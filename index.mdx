## Key Features

Powerful routing capabilities for modern PHP applications.

<CardGroup cols={2}>
  <Card
    title="HTTP Method Routing"
    icon="globe"
  >
    Support for GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS, and ANY methods
  </Card>
  <Card
    title="Dynamic Routing"
    icon="code"
  >
    PCRE-based pattern matching with regex support for complex routes
  </Card>
  <Card
    title="Flexible Callbacks"
    icon="function"
  >
